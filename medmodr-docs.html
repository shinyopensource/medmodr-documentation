Overview

MedModr is a free, web‑based statistical tool for mediation, moderation, serial mediation, and moderated mediation analysis. The tool provides unstandardized (b) and standardized (β) coefficients, bootstrapped confidence intervals for indirect effects, interactive path diagrams, and annotated simple slopes plots.

What MedModr Solves

🔗
Mediation AnalysisTest whether an intervening variable (M) transmits the effect of X on Y — simple and serial mediation.
⚡
Moderation AnalysisExamine whether the effect of X on Y depends on the level of a moderator W, with simple slopes plots.
🔄
Moderated MediationTest conditional indirect effects — whether mediation depends on a moderator (a‑path moderation).
📊
Bootstrap InferencePercentile bootstrap confidence intervals for all indirect effects (1,000–10,000 samples).

Simple Mediation (X → M → Y)

1
Select analysis type

In MedModr, click the Simple Mediation tab (X → M → Y).

2
Enter independent X, mediator M, outcome Y

Provide names and data for independent X, mediator M, and outcome Y. All three variables must have the same number of rows.

3
Add covariates (optional)

Enter control variables in the Covariates box: first line = names (space separated), subsequent lines = numeric values. Covariates are mean‑centered automatically.

4
Set options & run

Choose confidence level (90%, 95%, 99%), bootstrap samples (0 = Sobel only, 1000, 5000 or 10000).

Hayes (2009) recommend at least 5000 bootstrap samples for stable CIs of serial indirect effects. Article: Hayes, A. F. (2009). Beyond Baron and Kenny: Statistical Mediation Analysis in the New Millennium. Communication Monographs, 76, 408-420.

Click Run Analysis

5
Interpret results

The output includes: a‑path (X→M), b‑path (M→Y), total effect, direct effect, indirect effect (a×b), Sobel test, bootstrap CI, and interactive path diagram.

Serial Mediation (X → M1 → M2 → Y)

1
Select analysis type

Click the Serial Mediation button (X→M1→M2→Y).

2
Enter independent X, first mediator M1, second mediator M2, outcome Y

Provide names and data for independent X, first mediator M1, second mediator M2, and outcome Y. All four variables must have the same number of rows.

3
Add covariates (optional)

Enter control variables in the Covariates box: first line = names (space separated), subsequent lines = numeric values. Covariates are mean‑centered automatically.

4
Set options & run

Choose confidence level (90%, 95%, 99%), bootstrap samples (0 = Sobel only, 1000, 5000 or 10000).

Hayes (2009) recommend at least 5000 bootstrap samples for stable CIs of serial indirect effects. Article: Hayes, A. F. (2009). Beyond Baron and Kenny: Statistical Mediation Analysis in the New Millennium. Communication Monographs, 76, 408-420.

Click Run Analysis

5
Interpret results

Results include three specific indirect effects: X→M1→M2→Y (serial), X→M1→Y, X→M2→Y, plus total indirect, direct effect, and bootstrap CIs. The path diagram shows all coefficients (a₁, d₂₁, a₂, b₁, b₂, c′).

Moderation (X × W → Y)

1
Select analysis type

Click Moderation (X × W → Y).

2
Enter independent X, moderator W, outcome Y

Provide variable names and numeric data for independent X, moderator W, and outcome Y. All three variables must have the same number of rows.

3
Select mean‑centering strategy

Options: Mean Centering (All variables) (centers both numerical and binary variables [i.e., 0/1 or 1/0] before interaction), Mean Centering (Only continuous variables) (centers only numerical variables excluding binary variables - 0/1 or 1/0 before interaction), or No Mean Centering (raw multiplication).

4
Run & interpret interaction term

The table shows b₃ (X×W) coefficient with p‑value. If significant, moderation is present. Simple slopes at low (−1 SD), mean, and high (+1 SD) levels of W are displayed together with an annotated slopes plot. Each legend line includes the slope coefficient (b) and p‑value.

Moderated Mediation (Conditional Indirect Effect)

1
Select analysis type

Click Moderated Mediation tab (X → M → Y, with W moderating the a‑path).

2
Enter independent X, mediator M, moderator W, outcome Y

Provide variable names and numeric data for independent X, mediator M, moderator W, and outcome Y. All four variables must have the same number of rows.

3
Select mean‑centering strategy

Options: Mean Centering (All variables) (centers both numerical and binary variables [i.e., 0/1 or 1/0] before interaction), Mean Centering (Only continuous variables) (centers only numerical variables excluding binary variables - 0/1 or 1/0 before interaction), or No Mean Centering (raw multiplication).

4
Run & interpret interaction term

Output includes: simple slopes of X→M at three levels of W, conditional indirect effects (a·b) at each level, analytic and bootstrap CIs, and the index of moderated mediation (a₃·b)

Bootstrapping & Confidence Intervals

For all indirect effects (simple, serial, conditional, index), MedModr implements the percentile bootstrap. User‑selectable resamples: 1000, 5000, or 10000.

📊 Interpretation

If the bootstrap confidence interval (e.g., 95% BootLLCI to BootULCI) does not contain zero, the indirect effect is statistically significant at α = 0.05 (two‑tailed).

Standardized Coefficients (β)

Every regression table includes both unstandardized b and standardized β. Standardized coefficients are obtained by re‑running the same regression on z‑scored variables (mean = 0, SD = 1). This allows direct comparison of effect sizes across predictors measured in different units.

Path Diagrams & Simple Slopes Plots

📐
Interactive path diagrams
Click "View full‑screen diagram" to open a modal. Toggle between unstandardized/standardized coefficients, hide/show p‑values, download as PNG.
📈
Annotated slopes plot
For moderation / moderated mediation. Each slope line legend shows the slope (b) and p‑value. "Hide Labels" toggles annotations. Full‑screen mode + high‑resolution download.

Data Input Format

All data entry is text‑based: one numeric value per line, no header row. Example:

3.4
5.1
2.8
4.2

All variables must have the exact same number of rows (N). Covariates are optional: first line = variable names (space or comma separated), subsequent lines = numeric data.

📎 Covariates example
age education
25 16
30 18
28 17

Limitations

📐
OLS only — models are based on ordinary least squares regression; no multilevel or SEM.
🔢
Listwise deletion — rows with missing values must be removed before pasting data.
📦
Large N — performance may degrade with N > 5000 due to client‑side matrix inversion.
